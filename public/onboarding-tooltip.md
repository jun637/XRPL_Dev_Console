# **XRPL Dev Console에 오신 것을 환영합니다!**

이 사이트는 XRPL 개발자가 **네트워크 연결 → 지갑 관리 → 트랜잭션 구성/전송 → 결과 확인**의 XRPL 개발 파이프라인을 쉽게 테스트하고, 동시에 Flags·Transaction·Error Code 같은 자료로 기본 개념을 학습하며 온보딩할 수 있도록 만든 Dev Console입니다.  

**“XRPL을 처음 접한 사람도 바로 트랜잭션을 만들고 전송해 볼 수 있도록”** 하는 것이 목표입니다.

## 사용 방법이 궁금하다면?
스크린 왼쪽 상단의 사이드바 버튼을 열고, 사이드바 맨 위에 있는 **`사용법`** 버튼을 눌러주세요.  
자세한 단계별 안내와 참고 자료를 그곳에서 확인할 수 있습니다.  

**필요한 정보를 빠르게 확인하고, 바로 XRPL 개발을 시작해 보세요!**
